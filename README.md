# Task Tracker Web App

## Features
- User Login & Signup
- Add/Edit/Delete Tasks
- Priority Levels
- Search & Filter
- Dark Mode
- Colourful UI

## Tech Stack
- Flask (Python)
- SQLite
- HTML, CSS, JS
